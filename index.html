Model Reference

TC(Q,R) = c·μ + (K·μ)/Q + h·(Q/2 + R − μL) + (π·μ·n(R))/Q
μL = μ·L — mean lead-time demand
σL = σ·√L — std-dev of lead-time demand
n(R) = σL·[φ(z)−z·(1−Φ(z))] — exp. units short/cycle Step 0 → Q₀ = EOQ = √(2Kμ/h)
Step 1 → Φ(z) = 1−hQ/(πμ) ⟹ R = μL+z·σL
Step 2 → compute n(R)
Step 3 → Q = √(2μ·[K+π·n(R)]/h)
Repeat until |ΔQ| < ε

R = μL + z·σL  — reorder point Type 1 (Cycle SL):  F(R) = α  ⇒  z = Φ−1(α)
Type 2 (Fill rate):  n(R)/Q = 1−β  ⇒  Ψ(z) = Q(1−β)/σL
Safety stock: SS = z·σL
Ψ(z) = φ(z) − z·[1−Φ(z)]  — unit normal loss function
